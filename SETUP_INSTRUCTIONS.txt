================================================================================
                         GPU REQUIREMENTS
================================================================================

Recommended GPUs for RunPod:
1. RTX A6000 (48GB) - Best balance - ~$0.45/hr
2. A40 (48GB) - Great option - ~$0.40/hr
3. RTX 4090 (24GB) - Budget option - ~$0.35/hr
4. A100 (40GB) - High performance - ~$1.00/hr

The model needs ~20-25GB VRAM, so minimum 24GB GPU required.

================================================================================
                         COST ESTIMATION
================================================================================

Monthly cost for low-medium traffic (~1000 images/month):

- RunPod (serverless): $5-$20
  * Only pay when processing
  * ~$0.0006-$0.001 per image
  
- Vercel (frontend + backend): $0-$20
  * Free tier: 100 GB-hours
  * Usually sufficient for small-medium projects
  
- Total: ~$5-$40/month

For high traffic, consider:
- Increase RunPod max workers
- Upgrade Vercel plan ($20/month Pro)
- Use dedicated RunPod pods for better performance
